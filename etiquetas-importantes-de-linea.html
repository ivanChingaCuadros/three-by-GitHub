Etiqueta de linea

span

Es un contendedor de linea, equivalente a div, se suele usar para encerrar palabras o pequeños textos y darle estilo a través de CSS o localizarlos desde Js, semmáticamente no significa nada

Ejemplo

Lorem ipsum dolor sit,amet consectetur adipisicing elit. Commodi nesci unt, aspernatur voluptates rem fugit itaque molestiae excepturi illum debitis, impedit quae fuga odit nisi labore nulla ullam architecto odio repudiandae.

q

Es equivalente a blockquote, significa quote, siver para poner citas pero en linea

Ejemplo

La suerte nunca se olvida en versión Linea

code

Sirve para encerrar código que queremos representar visualmente, suele ir unido con la etiqueta pre

Ejemplo

            
                span {
                    color: red
                }
