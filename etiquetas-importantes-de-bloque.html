Etiquetas de bloque

address

Se utiliza para aportar información de contacto para el article más cercano o para todo el body, lo podemos poner en el footer como información relevante de una persona o un suceso

ejemplo

IVAN CHINGA
Madrid
28007
España

blockquote

Se utiliza para marcar las citas a otros autores o documentos, se puede incluir el atributo cite para poner un enlace al documento original o fuente, al clicar no te lleva a ese link, sino es mera información para el navegador

Tiene un equivalente que en lugar de bloque es de linea

Ejemplo

La suerte nunca se olvida

pre

Se utiliza para tener código preformateado que necesita ser representado igual que se escribió, se utiliza mucho para letras de canciones o un poema

Ejemplo

            hola  escribo en el lugar que me corresponde
                    o donde me salga de las naricez, pq 
             la vida es cruel y absurda
        o no?

div

Semánticamente no significa nada, se usa como división del documento, es un contenedor genérico que se usa generalemnte para dar estilos a través de CSS o para usar algo denominado deleganción de eventosen JavaScript

El contenido dentro de div esta relacionado, y si luego viene otro por ejemplo usuarios, es necesario abrir otro div

Ejemplo

Nombre1

email

Description

Nombre2

email

Description

hr

Horizontal rule, se utiliza para decirle al navegador que vamos a cambiar de tema
